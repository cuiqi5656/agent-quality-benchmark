## [0.1.0] - 2026-07-18

### Added

- Three agent touchpoints: validated trace upload, OpenAI-compatible fixture loop, and AQB HTTP v1.
- Evidence-first evaluation engine with category scoring, coverage rules, readiness gates, uncertainty, repeated-trial reliability, perturbations, and ablations.
- Thirty-six deterministic starter cases and two no-cost demo agents.
- Guided Next.js interface, run explorer, comparison, suite catalog, calibration queue, and JSON/CSV/HTML export.
- FastAPI/PostgreSQL/Redis/Celery deployment, immutable manifests, SSE progress, encrypted credential references, safe artifact storage, and deletion.
- Protocol schemas, migration, Docker Compose, tests, pinned CI, CodeQL, dependency audit, and container scanning.
